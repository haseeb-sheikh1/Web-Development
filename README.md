# Web Development Portfolio

This repository contains a collection of front-end web development projects. As a Software Engineering student with a foundational background in object-oriented programming and data structures, this repository serves as a practical application of transitioning those core concepts into modern web technologies. 

The projects act as a progressive documentation of my work with HTML, CSS, and JavaScript, focusing on responsive design, DOM manipulation, and application logic.

##  Repository Structure & Projects

Below is an overview of the projects included in this repository, along with the technical focus of each:

* **Music Player**
  * **Description:** A custom audio player web application. (still working on it)
  * **Key Features:** Implements audio playback controls, song sorting algorithms, and dynamic UI updates based on the current track state.

* **YouTube Clone**
  * **Description:** A front-end structural replica of the YouTube interface.
  * **Key Features:** Focuses on complex CSS grid/flexbox layouts, responsive design principles, and modern UI recreation without relying on external CSS frameworks.

* **Calorie Counter**
  * **Description:** A utility web application for tracking daily nutritional intake.
  * **Key Features:** Demonstrates form handling, user input validation, dynamic array management, and real-time mathematical calculations using JavaScript.

* **Rock, Paper, Scissors Game**
  * **Description:** A browser-based interactive game.
  * **Key Features:** Showcases event listener integration, core game logic implementation, and direct DOM manipulation for state rendering.

* **HTML-CSS**
  * **Description:** A dedicated directory for foundational styling exercises, layout experiments, and semantic HTML structuring.

##  Technologies & Tools

The current scope of these projects utilizes the core front-end stack:
* **HTML5:** Semantic markup and structure.
* **CSS3:** Styling, responsive layouts (Flexbox/Grid), and animations.
* **JavaScript (ES6+):** Application logic, data manipulation, and dynamic DOM interaction.
